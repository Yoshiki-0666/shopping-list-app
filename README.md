# 買い物リストアプリ v1

## 概要

よく買うものを登録し、今日買うものリストへ追加できる買い物リストアプリです。

## 使用技術

HTML / CSS / JavaScript / localStorage

## v1の目的

毎回商品名を入力する手間を減らす中心体験を検証するためのWeb版MVPです。
